Create, Retrive, Update Examples

Fetch students
The following is the response when you send a GET request to /students to fetch a list of all students in the database.
SQL: select s.id, s.firstName, s.lastName from student s

View student
The following is the response when you send a GET request to /students/id to fetch a single student record from the database.
SQL: select s.id, s.firstName, s.lastName, s.email from student s where s.id=:id

Update student
The following sends a PUT request to /students/id to update a single student record from the database.
SQL: select s.id, s.firstName, s.lastName, s.email from student s where s.id=:id
