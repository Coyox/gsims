1        Introduction

This section will contain an introduction for the document and software.

1.1 Intended readership

This document is intended for users of the GSIMS software, including but not limited to students, teachers, and administrative staff of Gobind Sarvar school[a].  No technical knowledge is assumed or required for readers of this document.  For a more technical overview, refer to “Installation Guide & Technical Information.”

1.2 Purpose

The purpose of this user manual is to assist the user in learning to use the functionalities of the GSIMS software system.

1.3 How to use this document

This section will provide an overview of the sections and which user types will find each section most useful.

1.4 Related Documents

This section will point to any other documents that may be of use to the user’s learning of this system.

For a more technical description, see “Installation Guide & Technical Information.”

1.5 Constraints

1.5.1 Technology Constraints[b]

  • As the application is web-based, access is limited by the user’s ability to connect to the internet.  Possible issues include but are not limited to: poor WiFi connectivity, slow or unstable local network, Internet Service Provider outage, and web browser failure.
  • The service is hosted off-site by a third party and is subject to the stability and performance of their systems.
  • All email functionalities will rely on an external email service.  If, at any point, the service is down while there is an attempt to send an email, the application will alert the user.
  • The application is designed to run on the following web browsers:
  • Google Chrome version 14 and higher,
  • Microsoft Internet Explorer version 9 and higher,
  • Mozilla Firefox version 7 and higher, and
  • Apple Safari version 7 and higher.

Performance on browsers other than those listed above, including older versions of those listed above, is not tested and no guarantee is made as to the functionality of the application except on the supported browsers.

  • JavaScript must be enabled on the user’s web browser; the application cannot be accessed when it is disabled.

2        General Functionality

This section will detail the functionality afforded to all users of the software. This section will also cover the procedures for registering for the system, signing in, and navigating the user interface. Will also re-iterate the supported platforms

2.1 Getting Started

2.1.1 Register for an account as a prospective student (staff are created internally and do not register on their own)

  1. From the home page, click the Register Today link.
  2. Registration Form: Fill out the information on the registration form (required fields indicated by red *), then press Next.
  3. From now on, the Previous button at the bottom of each page will navigate back to the previous page in the process.
  4. Course Selection:
  1. Select a school from the dropdown list, then a department.
  2. Click the plus icon next to the course you wish to enroll in to see the available sections for that course.
  3. Click the Enroll in this Section link next to the desired section to add it to the list for enrollment below.
  4. Click Remove in the table of enrolled sections to remove the section from the list.
  5. Note that a proficiency test may be required if selecting a course with pre-requisites (as listed in the prereqs column of the courses table).
  6. Press Next to continue.
  1. Terms and Conditions:
  1. Review the Terms and Conditions of Gobind Sarvar and select either Agree or Disagree.
  2. You will only be able to proceed if Agree is selected.
  3. Press Next to continue.
  1. Review the data you submitted and confirm that it is correct; press FINISH to complete registration.

2.1.2 Logging in

  1. From the home page, enter your username and password in the appropriate fields, then press enter on the keyboard or the Login button.

2.1.3 Reset forgotten password

  1. From the home page, click Forgot Password above the Login button.
  2. On the new page, enter the email address associated with your Gobind Sarvar account and press Submit to be sent a temporary link to reset the password for your account.

2.2 User Interface

2.2.1 Common Elements

The following elements are common to all logged in users of the system.

2.2.1.1 Logout Button

Located in the upper-right corner of every page, clicking the Logout button will end the user’s current session and return them to the login page.

2.2.1.2 User Info

Located in the upper-right of every page to the left of the Logout button is the current user’s username, their usertype (ie. student, teacher, admin, superuser), and the date they last logged in.

2.2.1.3 Help

At the bottom of the left-hand sidebar for every user is a Help feature.

5        Student Functionality

This section will cover the functionality most relevant to the student user class.

5.1 Function A

5.2 Function B - View Attendance History

  1. The Dashboard displays the student’s attendance for all enrolled classes over time.  Use the buttons above the table to navigate to other months.

5.2 Function B - View and Download Report Card (as CSV or PDF)

  1. The Dashboard displays the student’s report card for all enrolled courses.
  2. Use the Download as CSV and Download as PDF buttons to save the report card as a CSV or PDF file respectively.

5.2 Function B - Request Enrollment in a Course Section

  1. Select Course Enrollment from the sidebar.
  2. Select a school from the dropdown list, then a department.
  3. Click the plus icon next to the course you wish to enroll in to see the available sections for that course.
  4. Click the Enroll in this Section link next to the desired section to add it to the list for enrollment below.
  5. Click Remove in the table of enrolled sections to remove the section from the list.
  6. Note that a proficiency test may be required if selecting a course with pre-requisites (as listed in the prereqs column of the courses table).
  7. Press the Save Courses button at the bottom of the screen when finished.
  8. A message will be displayed confirming the action is successful, and that an administrator will process the enrollment request in the near future.

5.2 Function B - Send an Email

  1. Select the Send Email sidebar section.
  2. Enter the source email address, destination email address(es), Subject, and Message body in the appropriate fields.
  3. Click Preview HTML to see a preview of the email message with HTML encoding at the bottom of the page.
  4. Click Send to send the email; toggle the adjacent check box to optionally send a copy to the email address in the From field (to keep a copy for your own record).

5.2 Function B - Change Username or Password

  1. Select the Settings sidebar section.
  2. Enter new username and/or password (re-type the password a second time to confirm), then press Save to save changes.

6.1 Function A - Create a Student

  1. Enter first name, last name, and birthday; press Next.
  2. System will check if student is already in database
  3. If yes, ???
  4. If no, press Next.
  5. Fill out rest of student information (required fields indicated by red *), press Next
  6. Follow steps on Course Selection page (select school, department, course[s], section[s]); confirm selection(s) (press Remove if needed) and press Next
  7. Confirm information is correct; press FINISH

6.2 Function B - Edit Student Information

  1. Pull up the View Student view for the desired student.
  2. Under the Student Information tab, click Edit by the heading of the information that needs changing.
  3. Change the information as desired.
  4. Press Save to save the changes to the database, or press Cancel to discard the changes and leave the information as it currently is.
  5. A message will be displayed confirming the information was successfully updated.

6.3 Function C - Delete Student

  1. Pull up the View Student view for the desired student.
  2. Under the Student Information tab, click Delete Student.
  3. A message will be displayed confirming the student was successfully deleted.

6.4 Function D - Search for a Student

  1. Under the sidebar Students header, select Search Students.
  2. For a basic search, enter information in the boxes in the General Search area, and press Search to begin search or Clear to clear the information entered.
  3. For a more advanced search, enter information in the boxes in the Advanced Search area, and press Advanced Search to begin search or Clear to clear the information entered.
  4. Only one search (general or advanced) can be performed at once; they cannot be combined.
  5. A table will be returned with students matching the search parameters.

6.5 Function E - Browse all Students

  1. Under the sidebar Students header, select View all Students.
  2. The displayed table can be filtered using the search box, refreshed with the Refresh icon button, and exported using the Export icon button.
  3. Use the check box on each row to select students to email with the Email icon button; the check box in the column header can be used to toggle all students on or off at once.
  4. Students can be sorted by any of the first four columns (ID, First Name, Last Name, Email Address).
  5. Click View Student to go see detailed information and more functions for an individual student.

6.6 Function F - View and Modify a Student’s Enrolled Courses

  1. Pull up the View Student view for the desired student.
  2. Under the Courses tab, the student’s currently enrolled courses are displayed in the top table, and the courses they have taken in the past are displayed underneath.
  3. Click Drop Section to drop the student from a given Section.
  4. Click Add Section to enroll the student in another Section.

6.7 Function G - View a Student’s Attendance

  1. Pull up the View Student view for the desired student.
  2. Under the Attendance tab, the student’s attendance in all currently enrolled classes is displayed for the current month.
  3. Use the buttons at the top of the table to navigate to other months.

6.8 Function H - View and Download a Student’s Report Card

  1. Pull up the View Student view for the desired student.
  2. Under the Report Card tab, the student’s current grade is displayed for all currently enrolled courses.
  3. To download the report card in either CSV or PDF format, click the appropriate button at the top of the table (“Download as CSV” and “Download as PDF” respectively), then select a location to save the file.

6.9 Function I - Email a Student

  1. Pull up the View Student view for the desired student.
  2. Under the Email Student tab, compose and send an email as you would using the main Send Email tool from the sidebar. (see XXX for more details)

6.10 Function H - Search for a Teacher

  1. Under the sidebar Teachers section, select Search Teachers.
  2. Enter information (First Name and Last Name) and press Search.
  3. A table will be returned with teachers matching the search parameters.

6.11 Function D - Browse all Teachers

  1. Under the sidebar Teachers section, select View all Teachers.
  2. The displayed table can be filtered using the search box, refreshed with the Refresh icon button, and exported using the Export icon button.
  3. Use the check box on each row to select teachers to email with the Email icon button; the check box in the column header can be used to toggle all teachers on or off at once.
  4. Teachers can be sorted by any of the first four columns (ID, First Name, Last Name, Email Address).
  5. Click View Teacher to go see detailed information and more functions for an individual teacher.

6.12 Function D - Create a Teacher

  1. Under the sidebar Teachers section, select Create a Teacher.
  2. Enter the First Name, Last Name, and Email Address for the new teacher.
  3. Optional: enter the teacher’s course competency in the area below using the numbered buttons (0 indicates no ability to teach the course, 3 indicates ability to teach at the highest level, with 1 and 2 as intermediate levels).
  4. Press Save Teacher.
  5. A message will be displayed indicating the teacher was successfully created and an email will be sent to the address given with their login information.

6.4 Function D - View and Edit Detailed Teacher Information

  1. Pull up the View Teacher view for the desired teacher.
  2. Under the Teacher Information tab (open by default), the teacher’s information and course competency are displayed.
  3. Press Edit to modify any of the information.
  4. Press Save to make the changes to the database or Cancel to abort.
  5. A message will be displayed indicating the information was successfully updated.

6.4 Function D - Set a Teacher as an Administrator

  1. Pull up the View Teacher view for the desired teacher.
